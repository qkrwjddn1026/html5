미디어 관련 요소

이미지 5개 이상 - jpg, png, gif, webp, svg

동영상 3개 이상 - mp4, webm, ogv

  • video 요소의 태그 속성

    src: 해당 비디오 파일의 (위치와 파일명)

    poster : 대기화면(썸네일) 이미지의 파일 경로

    controls : 비디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복 재생

    width : 폭, height : 높이

    volum : 볼륨 크기 지정

    type : 해당 비디오나 오디오와 같은 미디어의 타입을 선언

    • 브라우저 마다 재생가능한 파일

    mp4, webm, ogv 파일이 브라우저 마다 각기 다르게 재생 가능 제공됨

    • audio 요소의 태그 속성

    src : 해당 오디오의 파일 경로

    controls : 오디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복 재생
